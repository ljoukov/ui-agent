# Iteration 1 Notes

## What changed
- Added baseline browser UX for the UKMT H6 teaching-agent example.
- Added live API-backed controls and real agent turn execution.
- Captured desktop and mobile screenshots for key user steps.

## Why
- Establish a real, runnable baseline and measure concrete UX issues using screenshots.

## What improved
- Functional end-to-end flow now exists in browser:
  - read prompt/problem
  - simulate/edit student attempt
  - run live Socratic teacher guidance

## What still needs work
- Typography and contrast are too weak.
- Action hierarchy and feedback visibility are poor.
- Mobile touch targets and spacing need substantial improvement.
- Too much on-screen information increases cognitive load.
