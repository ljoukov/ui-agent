# Iteration 3 Notes

## What changed
- Increased mobile-first typography and spacing.
- Added richer status feedback with spinner and live status region.
- Reworked action hierarchy (full-width primary run action).
- Improved guidance readability by chunking summary + bullets.
- Added auto-scroll to newly generated guidance.

## Why
- Address iteration-2 issues around readability, weak progress signaling, and dense guidance text.

## What improved
- Guidance is easier to scan and act on.
- Processing state is clearer while model is running.
- Primary action is more prominent and easier to tap.

## What still needs work
- Mobile action order and touch ergonomics still need refinement.
- Contrast for helper/disabled states can be stronger.
- Reset recovery pattern should be safer.
